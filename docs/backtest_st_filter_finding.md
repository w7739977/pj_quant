## 解读

### 1. 模型 70-80% alpha 是 ST 反弹

含 ST 累计 73.4% → 排 ST 累计 8.79% → **ST 贡献 +64.6pp = 88%**。

ST 股具备让模型「看起来很准」的几个特征：
- 涨跌幅 ±5% 限制 → momentum 因子捕捉「连续涨停」信号 → 训练时 forward return 标签经常 +25% (5 涨停)
- 流动性差 → 换手率因子异常活跃 → 模型识别为「热点」
- 低位反弹的统计规律性 → ML 容易拟合
- 跌停期间 forward 5d ret 高 → 训练标签偏向 ST

### 2. 真 α 扣成本后接近 0

```
ST 过滤池累计 α (16 个月):  +8.79%
单位换算: 年化 ≈ +6-7% (L2 假设，未扣成本)

A 股双边换手成本:  ~0.5-0.7% / 单边
5d hold 单次成本:  ~1% 双边
62 周 × 1%:         62% 成本

扣完后净 α ≈ 8.79% - 62% × 部分换手 = 大概率亏损
```

具体 net α 取决于实际换手率（不是每次推送都全换仓）。但**保守说，扣成本后 α 接近 0**。

### 3. 当前 22 因子模型在非 ST 池上的天花板

模型 R² = 0.0719（含 ST 训练）。如果重训在非 ST 池上：
- 可能 R² 下降（ST 提供的「强信号」消失）
- 也可能持平（模型本来就在非 ST 上学，ST 只是过拟合点）

但即使重训成功，**真 α 上限被因子库决定**。22 个因子（动量 + 波动 + 量价 + 基本面 + 财务）已经是当前天花板。

业界小盘策略「甜蜜点」在 50+ 因子（见 `optimization_backlog.md` P1.5）。

## 决策（路径 D — 接受现实）

| 选项 | 评估 | 选择 |
|---|---|---|
| A. 上 ST 过滤但不重训 | 累计 α 暴跌到 +9%，扣成本接近 0；客户看到效果变差 | ❌ |
| B. 维持现状（不动）| 数字漂亮但虚，含 ST 推送客户多半不能买 | ❌ |
| C. 上过滤 + 重训 | 可能 R² 下降；治标不治本（22 因子天花板） | ❌ 不值 |
| **D. 接受现实，转向因子库扩展** | 不再在 22 因子上做无效优化；P1.5 加 25+ 因子（北向 / 融资融券 / 龙虎榜 / 研报）后再评估 | **✅** |

## 给后人看的核心 lesson

> **本次发现是 VALIDATION_PHILOSOPHY 框架的实证案例。**

之前所有"参数优化"（NUM_POSITIONS 10/20、D_top3/5/10、3d vs d、rank 精选）都在**含 ST 数据**上做。一旦把 ST 移走，那些"优化"全部失效。

教训：**改参数前先验证 baseline 是否真有 edge**。当前 baseline 80% 是 ST 反弹机噪声，怎么调参数都是在调噪声。

下一阶段策略：**不调参，加因子**。
